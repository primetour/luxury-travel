## Sobre

Cada edição é um flipbook estilo Issuu com:

- PDF original em português preservado intacto
- Versão em inglês gerada com substituição tipográfica pixel-perfect
  (mesmas fontes, tamanhos, posições e alinhamento do original)
- Anúncios mantidos em português (conforme briefing editorial)
- Leitor responsivo: zoom, pinch, swipe, teclado
- Em celular: uma página de revista por vez (mag-page mode automático)
